## Getting Started

### 1. Ensure Virtual Environment is Activated

#### Linux/macOS:
```bash
cd langchain-langgraph-starter
source venv/bin/activate
cd Tutorial_11
```

#### Windows:
```cmd
cd langchain-langgraph-starter
.\venv\Scripts\activate
cd Tutorial_11
```

### 2. Launch Jupyter Notebook
```bash
jupyter notebook Tutorial_11_structured_data.ipynb
```
